# 🎓 Student Management System  
**Enrollment, Payment, and Records Management System**

## 📌 Overview  
The Student Management System (SMS) is a software application designed to streamline and automate academic and administrative processes in an educational institution.  

It integrates **student enrollment**, **payment tracking**, and **records management** into one centralized platform, improving efficiency, accuracy, and accessibility of data.

---

## 🚀 Features  

### 📖 Enrollment Management  
- Student registration and profile creation  
- Course/subject enrollment  
- Automated student ID generation  
- Enrollment status tracking  

### 💳 Payment Management  
- Tuition fee calculation  
- Payment recording and tracking  
- Partial and full payment support  
- Receipt generation  
- Balance monitoring  

### 🗂️ Records Management  
- Centralized student database  
- Academic records and grades  
- Attendance tracking (optional)  
- Document storage (e.g., transcripts)  
- Quick search and retrieval  

### 🔐 User Roles & Access Control  
- Admin dashboard (full access)  
- Staff-level permissions  
- Role-based authentication  

### 📊 Reports & Analytics  
- Enrollment statistics  
- Financial summaries  
- Student performance reports  
- Export to CSV/PDF
